Real-Time Weather App

Designed a website using HTML, CSS, Javascript and React framework, which tracks real-time weather of any city in the world. It provides additional real-time statistics on humidity, wind velocity, sunrise/sunset time, and ‘feel-like’ temperature of any city. It uses an API from openweathermap to fetch real-time data

Toilet Monitoring System

I designed a system to optimize cleaning frequency to solve the odor problem in school toilets. I built an ammonia detector that sounds an alarm when the ammonia concentration exceeds a certain threshold In order to check the statistics, I developed a website using HTML, CSS, and JavaScript along with Vue.js and Node.js. Used MongoDB for the database.
